Overview

Two Modes, One System

🛒

Market Mode

For vendor markets & pop-ups

  • ▸ QR code scan → mobile-first order form
  • ▸ Customer info collection (name, email, phone)
  • ▸ Magnet set selection with live pricing
  • ▸ Multi-photo upload with drag & drop
  • ▸ Payment method selection (Venmo, PayPal, Cash, Tap)
  • ▸ Auto sales tax calculation (8.25%)
  • ▸ Venmo & PayPal deep links with total pre-filled
  • ▸ Order confirmation with instructions per method
  • ▸ Photos auto-uploaded to Google Drive by order #
📸

Event Mode

For weddings, parties & events

  • ▸ QR code scan → instant camera or upload
  • ▸ In-app camera capture (snap a photo live)
  • ▸ Upload from gallery (existing photos)
  • ▸ Auto-upload to event-specific Google Drive folder
  • ▸ No login, no ordering — zero friction for guests
  • ▸ Thank you confirmation with event branding
  • ▸ Admin panel to create events & generate QR codes
  • ▸ Bulk photo gallery view per event

Admin Dashboard

A real-time order management dashboard where you can view and manage all incoming orders — no refreshing needed.

Order Queue

Live feed of incoming orders with customer name, set type, qty, total, payment method, and status.

Photo Preview

Thumbnail previews of uploaded photos per order. Click to open full-res in Google Drive.

Google Drive Sync

Photos auto-organized into folders by order number. Zero manual file management.

Payment Tracking

Mark orders as paid/unpaid. Filter by payment method. Daily sales summary.

Event Manager

Create events, set dates, generate unique QR codes, and view all uploaded photos per event.

QR Code Generator

Generate printable QR codes for market mode or event mode — each with its own branded landing page.

Technical Architecture

Built for speed and reliability with no ongoing server costs.

Frontend

Single-page app (HTML/JS/CSS)

Mobile-first, hosted on GitHub Pages — zero hosting cost.

Backend

Firebase (Firestore + Auth)

Real-time order database with admin authentication. Free tier covers high volume.

File Storage

Google Drive API

Photos uploaded directly to Drive, auto-organized by order # or event name.

Payments

Venmo & PayPal Deep Links

Pre-filled payment links with amount. Cash & tap handled in person.

Camera

WebRTC / MediaDevices API

In-browser camera capture for event mode — no app install required.

Notifications

Email + Real-time alerts

Order confirmation emails to customers. Live push alerts on admin dashboard.
